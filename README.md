# 🤖 Basic Rule-Based Chatbot (Python)

## 📌 Description

This is a simple console-based chatbot built using Python.
It responds to user queries based on predefined rules and keyword matching.

## 🚀 Features

* Greets user based on time (Morning/Afternoon/Evening)
* Responds to basic queries based on predefined rules
* Uses dictionary-based response system
* Simple and beginner-friendly logic
* Includes response delay for better interaction

## 🛠️ Technologies Used

* Python
* Dictionary & Functions
* Loops & Conditional Statements

## ▶️ How to Run

```bash
python main.py
```

## 📷 Example

User: hello
Bot: hello Mohit, How can I help you?
